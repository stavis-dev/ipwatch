
# Инструкция по сборке ipwatch

Проект написан на чистом Go и не имеет внешних зависимостей (`zero-dependencies`), что делает его сборку максимально простой. Вам не нужно скачивать сторонние пакеты.

## Требования
* Установленный Go (версии 1.16 или выше).

---

## Быстрый запуск для разработки

Вы можете запустить утилиту напрямую из исходного кода без компиляции бинарного файла:

```bash
go run main.go -t

```

---

## Компиляция для текущей платформы

Чтобы собрать исполняемый файл для операционной системы, на которой вы сейчас находитесь, выполните:

```bash
go build -o ipwatch main.go

```

*На Windows команда создаст файл `ipwatch.exe`.*

---

## Кросс-компиляция (Сборка под другие ОС)

Go позволяет собирать бинарники для Windows, Linux и macOS с любой платформы без настройки сложного окружения. Для этого используются переменные `GOOS` (операционная система) и `GOARCH` (архитектура процессора).

### Сборка для Windows (64-bit)

Вы можете запустить эту команду на Linux или macOS, и на выходе получите готовый `.exe` файл для Windows:

```bash
GOOS=windows GOARCH=amd64 go build -o ipwatch.exe main.go

```

### Сборка для Linux (64-bit)

```bash
GOOS=linux GOARCH=amd64 go build -o ipwatch main.go

```

### Сборка для macOS (Apple Silicon - M1/M2/M3)

```bash
GOOS=darwin GOARCH=arm64 go build -o ipwatch main.go

```

### Сборка для macOS (Intel)

```bash
GOOS=darwin GOARCH=amd64 go build -o ipwatch main.go

```

---

## Оптимизация размера файла (Финальная сборка)

По умолчанию Go добавляет в бинарник отладочную информацию. Если вы хотите сделать файл максимально легким (уменьшить размер примерно на 30-40%), собирайте его с флагами оптимизации `-ldflags="-s -w"`:

```bash
# Пример для Windows
GOOS=windows GOARCH=amd64 go build -ldflags="-s -w" -o ipwatch.exe main.go

# Пример для Linux
GOOS=linux GOARCH=amd64 go build -ldflags="-s -w" -o ipwatch main.go

```

* Флаг `-s` убирает символы отладки (debug информацию).
* Флаг `-w` отключает генерацию DWARF-таблиц.

```
